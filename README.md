# MoodBoost

![MoodBoost Screenshot](src/images/screenshot.png)


**MoodBoost** is a personalized wellness platform that curates music, books, images, videos, and motivational quotes based on your current mood. The site helps enhance emotional well-being by providing mood-specific content, whether you're feeling relaxed, energized, or introspective.

## Features
- **Mood-Based Music**: Discover playlists that match your emotional state.
- **Curated Book Recommendations**: Get book suggestions to uplift or relax you based on your mood.
- **Motivational Quotes**: Receive mood-specific quotes for inspiration and encouragement.
- **Mood-Themed Images & Videos**: Explore images and videos tailored to your emotional needs.

## Installation

### Prerequisites
Before running the project, make sure you have the following installed:
- **Node.js** (v14 or later)
- **npm** (v6 or later)

### Steps to Set Up Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/moodboost.git
   ```
   
2. Navigate to the project directory:
   ```bash
   cd moodboost
   ```

3. Install the required dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm start
   ```

5. Open your browser and go to `http://localhost:3000` to view the application.

## Usage
Upon visiting the website, users are prompted to select their current mood. Based on their selection, the site will display personalized content, such as:
- Music playlists to match their mood.
- Book recommendations tailored to their emotional state.
- Inspirational quotes that fit their mood.
- Motivational images and videos for encouragement.
